🍽️ Rapidex

Rapidex é um aplicativo web de receitas rápidas onde o usuário pode escolher pratos com base na proteína principal.

🚀 Funcionalidades

Filtro por proteína (Frango, Carne, Peixe, Ovo)

Visualização de receitas

Botão Ver mais com ingredientes e modo de preparo

Menu responsivo

Modo escuro

Layout responsivo

🛠️ Tecnologias utilizadas

HTML5

CSS3

JavaScript

jQuery

Font Awesome

📱 Interface

O usuário pode:

Navegar pelo cardápio

Filtrar receitas

Visualizar ingredientes e preparo

Marcar receitas favoritas

📂 Como executar o projeto

Baixe ou clone o repositório

Abra o arquivo index.html no navegador
